Yeehaw!

Icons are generated from a catalog definition file: icons/icon-pack.json

You can use glob file patterns to include SVG files in the generated output. An index file that lists all included input files is written to: dist/v{x}.{y}.{z}/icons/index.json

For the time being, the task just copies those SVG files from icons folder to dist folder. You could, however, add a SVG-to-Webfont plugin such as gulp-fontcustom, gulp-svgicons2svgfont, gulp-iconfont, or gulp-webfont.
